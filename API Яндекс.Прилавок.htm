Main.Kits - Получение всех наборов

  • сравнить с:
  • 1.0.0

Эндпоинт для полуучения всех наборов внутри конкретной карточки ИЛИ наборов, созданных пользователем.

  • Обязательно нужно передать заголовок Authorization ИЛИ параметр cardId, для получения наборов

  • Если на вход приходит запрос с заголовком Authorization, содержащим authToken конкретного пользователя - будут возвращены наборы, созданные этим пользователем.

  • Если на вход приходит query-параметр cardId, буду возвращены наборы, относящиеся к данной карточке.

  • В случае, если ни один из параметров не передан - вернётся ошибка.

  • В случае, когда переданы оба параметра - Authorization является приоритетным

get 
/api/v1/kits

Header

Название Тип Описание
Authorization необязательный string

Авторизационный заголовок в формате Bearer {authToken}. При передаче - возвращает все наборы, созданные пользователем.
  • Получение наборов, созданных пользоваталем 
{
    "Authorization": "Bearer jknnFApafP4awfAIFfafam2fma"
}

Main.Kits - Создание набора

  • сравнить с:
  • 1.0.0

Эндпоинт для создания набора внутри конкретной карточки ИЛИ пользователя.

  • Обязательно нужно передать заголовок Authorization ИЛИ параметр cardId, для создания набора

  • Если на вход приходит запрос с заголовком Authorization, содержащим authToken конкретного пользователя - будет создан набор лично пользователя

  • Если на вход приходит параметр cardId, будет создан набор внутри соответствующей карточки

  • В случае, если ни один из параметров не передан - вернётся ошибка.

  • В случае, когда переданы оба параметра - Authorization является приоритетным

post 
/api/v1/kits

Header

Название Тип Описание
Authorization необязательный string

Авторизационный заголовок в формате Bearer {authToken}. При передаче - возвращает все наборы, созданные пользователем.
Content-Type необязательный string

По умолчанию: application/json
  • Пример заголовков
  • Получение наборов, созданных пользоваталем 
{
    "Content-Type": "application/json"
}
{
    "Content-Type": "application/json",
    "Authorization": "Bearer jknnFApafP4awfAIFfafam2fma"
}

Параметр

Название Тип Описание
cardId необязательный number

id карточки в таблице card_model. При передаче - создаст набор, входящий в состав данной карточки
name string

name набора, который будет записан в соответствующее поле таблицы kit_model.
